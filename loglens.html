LogLens

Desktop Log Analyzer (PySide6 + SQLite)

← Back GitHub

LogLens is a lightweight desktop tool that ingests log files into a local SQLite database for fast filtering, searching, and exporting. Built to resemble real internal tooling workflows without using proprietary data.

Highlights

  • Load multiple log files and parse common formats (timestamp/level/source + JSON log lines).
  • Filter by level and search across message/source/raw/file.
  • Sortable table view for quick investigation.
  • Export filtered results to CSV.
  • Double-click any row to view the full raw log line (with Copy).

Architecture

  • Parser: normalizes lines into structured fields (timestamp, level, source, message, raw).
  • Storage: SQLite table for fast filtering and exporting.
  • UI: PySide6 table model + debounced filters.
  • Async ingest: parsing runs in a worker so the UI stays responsive.

What I’d Add Next

  • Charts (errors over time, top sources).
  • Saved filters / searches.
  • Rule-based highlighting and “incident bundle” export.
