LogLens

Desktop Log Analyzer (PySide6 + SQLite)

← Back GitHub

LogLens is a lightweight desktop tool that ingests log files into a local SQLite database for fast filtering, searching, and exporting.

Highlights

  • Load multiple log files and parse common formats (timestamp/level/source + JSON log lines).
  • Filter by level and search across message/source/raw/file.
  • Sortable table view for quick investigation.
  • Export filtered results to CSV.
  • Double-click any row to view the full raw log line (with Copy).

What it demonstrates

  • UI development with Qt (PySide6)
  • Parsing + normalization of unstructured data
  • Local database usage (SQLite) for fast filtering
  • Usability touches: debounced filters, export, raw-line inspection
